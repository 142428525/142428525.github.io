# 注记
## `layout`设置
- `MGE`只应当在`/MGE/`下使用
- `news`只应当在`/news/`下使用
- `MGE_style`不作限制
## magic numbers
### `MGE`
- `1` bool，是否激活第1个导航链接框（主页）
- `2` bool，是否激活第3个导航链接框（联系）
- `3` bool，是否激活第4个导航链接框（关于）
### `news`
- `1` bool，是否激活第1个导航链接框（主页）
- `2` bool，是否激活第3个导航链接框（特辑）
